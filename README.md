# LM Profesionales - Blog Web con Django

## 📌 Descripción

LM Profesionales es una aplicación web desarrollada en Django que simula un blog profesional orientado a servicios contables, financieros y análisis de datos.

La plataforma permite la gestión de publicaciones, autenticación de usuarios, perfiles personalizados y un sistema de mensajería interna entre usuarios.

---

## 🚀 Funcionalidades

### 🔹 Generales
- Página de inicio (Home)
- Sección "Acerca de mí" (About)
- Navegación mediante Navbar

### 🔹 Blog / Pages
- Listado de publicaciones
- Visualización completa de contenido
- Creación, edición y eliminación de páginas (requiere login)
- Mensaje en caso de no existir publicaciones

### 🔹 Autenticación
- Registro de usuario (Signup)
- Inicio de sesión (Login)
- Cierre de sesión (Logout)

### 🔹 Perfil de Usuario
- Visualización de datos personales
- Edición de perfil
- Avatar
- Biografía

### 🔹 Mensajería
- Envío de mensajes entre usuarios
- Bandeja de entrada (Inbox)
- Contador dinámico de mensajes en el Navbar

---

## 🛠️ Tecnologías utilizadas

- Python
- Django
- SQLite

- CKEditor (texto enriquecido)

## ⚙️ Instalación y ejecución

1. Clonar el repositorio:
git clone https://github.com/crnahuellara-maker/tu-repo.git


2. Crear entorno virtual:
python -m venv env


3. Activar entorno:
env\Scripts\activate

4. Instalar dependencias:
pip install -r requirements.txt

5. Migraciones:
python manage.py migrate

6. Ejecutar servidor:
python manage.py runserver
